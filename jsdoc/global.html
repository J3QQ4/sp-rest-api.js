SpRestApiOptions

The options passed to methods inside the SpRestApi class.
Type:
  • Object
Properties:
Name Type Attributes Description
expand string | Array.<string> <optional>
 The columns to be expanded, e.g. lookup columns or people/group columns. Use internal column names instead of display names.
filters string | Array.<string> <optional>
 The filters to be used in filtering the list items using the $filter parameter. Can be a string like "(Project eq 'Project 1') or (Project eq 'Project 2')", or an array of strings like ["Project eq 'Project 1'", "Id eq 18"], each of which will be concatenated automatically in the filter string using the AND condition. To use OR condition, build the string manually.
listTitle string <optional>
 The display name of the SharePoint list.
maxItems number <optional>
 The maximum number of items to be returned from the list. If `recursiveFetch` is set to true, this is the maximum number of items to fetch on each request to server. If not not specified, defaults to SharePoint's limit of 100. Maximum is 5000 due to SharePoint limitations.
onsuccess function <optional>
 The callback function for successful requests to SharePoint REST API.
onerror function <optional>
 The callback function for failed requests to SharePoint REST API.
recursiveFetch boolean <optional>
 Fetch all items from the list by repeatedly making server requests until all list items are fetched. This is to overcome SharePoint's limitation of maximum 5000 items per call.
select string | Array.<string> <optional>
 Which columns to select from a list (instead of fetching all columns). If the value is string, it must be in the format 'Id,Title,Status,CompletionDate,Approver/Title'. If the value is an array, it must be an array of strings like ['Id', 'Title', 'Status', 'CompletionDate', 'Approver/Title'].
siteUrl string <optional>
 The SharePoint site URL which is usually obtained from the _spPageContextInfo.webAbsoluteUrl. Required if using this library outside of a SharePoint page.
token string <optional>
 The SharePoint's request digest, a token that it is required for every API call. Required only if using outside a SharePoint page, otherwise it is obtained automatically from DOM.
verbosity Verbosity <optional>
 The amount of metadata to be returned in the JSON response from server. Use the SpRestApi.Verbosity enum.
urls Array.<string> <optional>
 The URLs of various API calls, e.g. to get a list item, all items in a list etc.
